# Job Description: Founding Full Stack Engineer (SparqAI)

**Position:** Founding Full Stack Engineer / Lead Developer  
**Location:** Remote / Hybrid (Specify)  
**Type:** Full-time / Contract (Specify)  

## About SparqAI
SparqAI is an early-stage SaaS startup building an **ROI Decision Intelligence Platform**. We help businesses measure, analyze, and optimize their return on investment through advanced data analytics and intelligent decision support. We are looking for a hands-on engineer to take ownership of our MVP and help us scale to our first pilot customers.

## The Role
We are seeking a **Founding Full Stack Engineer** who is capable of building and maintaining a complete web application from scratch. You will effectively be the lead engineer, working directly with the founder to turn the "Plan of Action" into a deployed, functional product. You will be responsible for the full lifecycle of development—from database design and API development to frontend UI implementation and deployment.

## Key Responsibilities
*   **Full Stack Development:** Build and maintain the core platform using **FastAPI (Python)** for the backend and **Next.js (TypeScript/React)** for the frontend.
*   **Data Integration:** Implement and manage integrations with third-party data sources (e.g., **HubSpot**, CRM systems) to ingest ROI-related data.
*   **Database Management:** Design and optimize database schemas using **PostgreSQL** and **SQLAlchemy**.
*   **Analytics & Visualization:** Build interactive dashboards and charts using libraries like **Recharts** to visualize complex data for decision-making.
*   **DevOps & Infrastructure:** Manage the containerized environment using **Docker** and **Docker Compose**. Ensure smooth deployment and CI/CD pipelines.
*   **Product Ownership:** Collaborate on product requirements, provide technical feasibility assessments, and iterate rapidly based on user feedback.

## Requirements
*   **Backend:** Strong proficiency in **Python 3.11+**, with deep experience in **FastAPI** and **SQLAlchemy**.
*   **Frontend:** Solid expertise in modern **JavaScript/TypeScript**, specifically **React 19** and **Next.js 16 (App Router)**.
*   **Database:** Experience with **PostgreSQL**, including schema design and performance optimization.
*   **UI/UX:** Ability to implement pixel-perfect, responsive designs using **Tailwind CSS**. Experience with glassmorphism or modern premium aesthetics is a plus.
*   **Infrastructure:** Comfortable with **Docker**, Docker Compose, and basic Linux server management.
*   **Problem Solver:** Ability to work autonomously, debug complex issues across the stack, and make architectural decisions.

## Nice to Have
*   Experience in the **AdTech**, **MarTech**, or **Business Intelligence** space.
*   Familiarity with Data Engineering (e.g., Pandas, NumPy) or basic ML concepts.
*   Experience integrating with marketing platforms (Google Ads, LinkedIn Ads, HubSpot).

## Tech Stack
*   **Backend:** Python, FastAPI, Pydantic, SQLAlchemy, PostgreSQL
*   **Frontend:** TypeScript, Next.js 16, React 19, Tailwind CSS 4, Recharts
*   **Infrastructure:** Docker, Docker Compose

## Why Join Us?
*   Opportunity to build a product from the ground up and have a significant impact on technical direction.
*   Work on solving high-value business problems (ROI intelligence).
*   Agile, fast-paced environment focused on shipping and iterating.
